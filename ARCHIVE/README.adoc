== Purpose

The ARCHIVE/ directory preserves old versions for:

* **Historical reference** - Understanding how the license evolved
* **Legacy support** - Supporting projects using older versions
* **Research** - Academic study of license development
* **Compliance** - Legal requirements to maintain version history
