# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.0.

The Palimpsest‑MPL License ecosystem is built on principles of care, ethical use, emotional lineage, and provenance integrity.  
All contributors, maintainers, and community members are expected to uphold these values in addition to the standards defined in the Contributor Covenant.

## Our Code of Conduct

This project adopts the Contributor Covenant Code of Conduct, version 2.0:

https://www.contributor-covenant.org/version/2/0/code_of_conduct/

Translations are available at:

https://www.contributor-covenant.org/translations

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported privately to the Palimpsest Stewardship Council.  
Contact information is available in the repository README.

All reports will be handled promptly, respectfully, and in accordance with the governance processes defined in:

- `GOVERNANCE.adoc`
- `MAINTAINERS.adoc`
- `CODE_OF_PRACTICE.adoc`

The Council is responsible for clarifying and enforcing standards of acceptable behavior.
