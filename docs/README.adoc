= Documentation Index – Palimpsest‑MPL Ecosystem
:toc: macro
:toclevels: 2

This directory contains contributor‑facing documentation for understanding, integrating with, and navigating the Palimpsest‑MPL ecosystem.  
It complements the legal, governance, and specification layers by providing conceptual maps, onboarding materials, diagrams, and integration guides.

toc::[]

== Purpose

The `docs/` directory serves as the knowledge base for contributors.  
It provides:

* onboarding pathways  
* conceptual and narrative maps  
* integration guidance  
* provenance flow diagrams  
* tooling documentation  
* jurisdictional comparisons
