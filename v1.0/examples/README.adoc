== Quick Start

Each example includes:

* Proper LICENSE file
* SPDX headers in source files
* Package manager configuration
* README with usage instructions
* Optional: Provenance metadata

**To try an example:**

[source,bash]
----
# Rust example
cd v1.0/examples/rust && cargo build

# Julia example
cd v1.0/examples/julia && julia example.jl

# Deno example
cd v1.0/examples/javascript && deno run index.js
----

== Common Patterns

=== Minimal PMPL Project

[source,text]
----
my-project/
├── LICENSE               ← PMPL-1.0 text
├── README.md             ← Project documentation
└── src/
    └── main.rs           ← SPDX headers
----

=== PMPL with Provenance

[source,text]
----
my-project/
├── LICENSE
├── LINEAGE.md            ← Emotional lineage documentation
├── provenance/           ← Quantum-safe signatures
│   └── signatures.json
└── src/
    └── main.rs           ← SPDX headers + provenance metadata
----

=== Multi-License Project

[source,text]
----
my-project/
├── LICENSE               ← Multi-license notice
├── LICENSES/             ← Individual license texts
│   ├── PMPL-1.0-or-later.txt
│   └── MIT.txt
└── src/
    ├── pmpl-code.rs      ← SPDX: PMPL-1.0-or-later
    └── mit-code.rs       ← SPDX: MIT
----
